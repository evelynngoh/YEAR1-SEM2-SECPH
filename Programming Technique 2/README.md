Programming Technique II course provided valuable understanding and practical experience in object-oriented programming (OOP) using the C++ programming language. Throughout the course, important OOP concepts such as encapsulation, inheritance, polymorphism, associations, and object manipulation were explored through lectures, exercises, laboratory activities, and project-based learning. The hands-on programming tasks and mini project strengthened problem-solving, analytical thinking, and coding skills while improving the ability to design and develop medium-scale software applications using object-oriented approaches. In addition, teamwork and collaboration skills were enhanced through group project activities, discussions, and GitHub-based teamworking assessments. Exposure to UML class diagrams, software design techniques, and advanced C++ features also increased understanding of structured software development practices in real programming environments. Overall, this course successfully enhanced both theoretical understanding and practical competency in object-oriented programming and software development using C++.

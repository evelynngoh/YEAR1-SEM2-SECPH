System Analysis and Design course provided valuable understanding of system development methodologies, project planning, system analysis, and system design processes used in real-world information systems development. Throughout the course, knowledge of project feasibility studies, project management, data flow diagrams (DFD), user interface design, system implementation, and object-oriented analysis and design was strengthened through lectures, quizzes, laboratory work, and project-based learning activities. The industry-based group project enhanced practical skills in analysing real organizational problems, gathering system requirements, designing system models, and developing suitable system solutions using both structured and object-oriented approaches. In addition, teamwork, communication, leadership, and problem-solving skills were improved through collaborative project development, presentations, peer assessments, and industry evaluation activities. Exposure to Agile tools, modelling techniques, UML diagrams, and real-world case studies also increased understanding of how systematic planning and design contribute to successful software and information system development. Overall, this course successfully enhanced both theoretical understanding and practical competency in system analysis, system design, and project management within software development environments.

Probability and Statistical Data Analysis course provided valuable understanding of statistical concepts, data analysis techniques, and probability models used in analysing real-world data. Throughout the course, knowledge of descriptive statistics, probability distributions, hypothesis testing, chi-square analysis, correlation, regression, and ANOVA was strengthened through lectures, assignments, quizzes, laboratory sessions, and project-based learning activities. Practical exposure to statistical analysis using R programming software enhanced analytical thinking, problem-solving, and data interpretation skills while improving the ability to analyse and present data in a meaningful manner. In addition, the group projects and presentations strengthened teamwork, communication, and critical thinking skills through collaborative data collection, statistical analysis, and report preparation. The course also increased understanding of how statistical methods support data-driven decision-making and research analysis in various academic and industrial applications. Overall, this course successfully enhanced both theoretical understanding and practical competency in probability, statistics, and data analysis techniques.

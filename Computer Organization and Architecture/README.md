Computer Organization and Architecture course provided valuable understanding of the fundamental concepts of computer hardware, system architecture, and low-level programming. Throughout the course, knowledge of computer components, data representation, arithmetic operations, instruction set architecture, CPU operations, memory systems, pipelining, and input/output systems was significantly strengthened through lectures, tutorials, laboratory exercises, and project-based learning activities. The hands-on assembly language programming tasks using MS Visual Studio enhanced problem-solving, logical thinking, and programming skills while providing practical exposure to low-level machine operations and instruction execution. In addition, the group project and presentation activities improved teamwork, communication, and technical reporting skills through collaborative system analysis and implementation tasks. The course also increased understanding of how computer hardware and software interact to achieve efficient system performance in modern computing environments. Overall, this course successfully enhanced both theoretical knowledge and practical competency in computer organization, architecture, and low-level system programming.
